# content-automation-engine
Content Automation Engine built with Python. Generates titles, descriptions, hashtags, and engagement comments for gaming content creators.

# Estructura de games.json

## plataforma
Plataforma principal del videojuego.

Ejemplos:
- PS1
- PS2
- SNES

## genero
Género principal.

Ejemplos:
- Plataformas
- Carreras
- RPG

## dificultad
Dificultad general del juego.

Valores:
- Baja
- Media
- Alta
